CVE 취약점 진단 자동화
& 인벤토리 관리 솔루션

SolidStep CVE

CVE 취약점 진단부터 권고, 조치 결과 확인,보고까지
One-Stop으로 관리합니다.

자료다운로드
CVE 취약점 진단 + 국내 시장에 맞게 다양한 기능 추가

SolidStep CVE는 국내 시장에 맞게 다양한 기능을 추가하여 CVE 취약점 진단 뿐만 아니라 자산별 상세 인벤토리 관리까지 가능한 CVE 취약점 지니단 및 인벤토리 관리 솔루션 입니다.
최신 발표된 취약점 정보를 주기적으로 반영하여 신규 취약점에 신속한 대응이 가능하며, CVE 취약점 진단부터 권고, 조치 결과 확인, 보고까지 모든 과정을 체계적으로 관리할 수 있습니다.

SolidStep CVE, 드리는 약속

SolidStep CVE는 주기적인 최신 CVE 취약점 정보 반영으로 빠른 신규 취약점 대응이 가능합니다.
에이전트, 원격 진단(Agentless), 수동 진단 방식을 지원하며 CVSS 버전별(V2, V3)진단 및 버전별 전환이 가능합니다.
자산별 취약점 현황 및 상세 인벤토리 관리가 가능하여 전체 자산에 대한 쉽고 빠른 원클릭 전수 진단을 실행하며 자산관리대장 작성이 용이하고, 자산 및 인벤토리 관리가 탁월합니다.

완벽 대응

최적화된 아키텍처
SolidStep CVE 구성도

SolidStep CVE는 에이전트, 원격 스캔, 수동 진단 방식을 지원하여 고객 맞춤 진단이 가능하며,
최신 CVE 정보를 주기적으로 업데이트 반영하고 최상의 진단 구성을 갖추고 있습니다.

SolidStep CVE 기대효과

SolidStep CVE와 함께라면 자동화된 CVE 취약점 진단 및 자산 관리로 가시성 확보하고 지속적인 CVE 취약점 평가로
보안 위협과 사고 사전 예방 및 보안 진단 프로세스 개선으로 신속하고 체계적인 보안 관리가 가능합니다.

편리한 웹 UI 방식의 통합 대시보드
  • 사용자 접근성과 편의성을 고려한 Web UI 방식의 대시보드 제공으로 전체 자산에 대한 현황 및 상세정보, 인벤토리, CVE, 진단현황, 보고서, 조치 관리 등 다양한 현황 파악 및 관리 용이
  • 보안성 평가를 위한 점수 산정 기준(CVSS v2, v3)전환 기능 제공
자산·인벤토리 관리
  • 자산별,운영 부서별 다양한 그룹핑(연결그룹)설정으로 그룹화 관리 가능
  • 관리자별, 사용자별 접근 권한 관리 가능
  • CVE 취약점 점수 및 정보 제공
  • 인벤토리별 CVE 취약점 상세 정보, 보유 자산내역 및 상세 정보 확인 가능
상세한 진단 결과 보고서
  • 취약점 진단 항목의 한글화 전환 기능 제공
  • 취약점 진단 후 웹 UI 보고서 및 엑셀 형식의 다양한 결과 보고서
    (개별,그룹,전체)제공
신속한 취약점 조치 관리
  • 승인이 필요한 작업 요청에 대한 결재선을 지정함으로써 신속한 취약점 조치 계획 수립 및 승인 가능
  • 취약점 처리 내역 현황 및 결재 요청 리스트에 대한 효율적인 이력 관리 가능
