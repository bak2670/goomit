이메일 위협을 완벽 제거

모든 이메일 링크와 첨부 파일은 변환되어 격리 플랫폼을 통해 전달됩니다.
이를 위한 작업에 별도의 애플 라이언스 또는 엔드 포인트 클라이언트, 에이전트나 플러그인이 필요하지 않습니다.
격리 기술을 활용한 Menlo Security EMAIL은 사용자의 이메일 경험을 보존하고 생산성을 그대로 보호합니다.

Menlo Security EMAIL, 드리는 약속

Menlo Security EMAIL은 이메일 본문 내 포함된 URL의 안전한 격리 처리와 첨부 파일에 대한
격리 실행 및 세이프 PDF(Safe PDF) 변환 처리를 통하여 사용자의 안전한 이메일 사용 환경을 지원합니다.

Menlo Security 구성도

Menlo Security는 단일 클라우드를 이용합니다. 사용자 위치에 상관없는 100% 안전한 웹 보안 서비스를 제공합니다.

Menlo Security EMAIL 기대효과

Menlo Security EMAIL은 모든 이메일 링크와 첨부파일은 변환되어 격리 플랫폼을 통해 전달합니다.
이메일 본문에 포함된 URL 위협에 대해 보안하는 유일한 솔루션으로 지메일(Gmail), 오피스 365, Exchange Server 등 다양한 이메일 사용 환경에 대한 완벽 지원합니다.

이메일 본문 URL은 안전한 격리 플랫폼에서 실행
위험도가 높은 웹페이지는 Read-Only 실행
첨부파일 확인 시 Safe PDF 변환
격리 기술로 악성코드 원천 차단
필요시 원본 첨부파일 다운로드 권한 설정 가능
클라우드 기반 AV, SandBox, DLP 제공
