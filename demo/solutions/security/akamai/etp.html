Akamai SIA

세계 최대 규모의 엣지 플랫폼에서 100% 가용성의 웹 보안을 제공합니다.

자료다운로드
제로데이 멀웨어 및 피싱을 선제적으로 방어합니다

Akamai SIA(Enterprise Threat Protector)는 기업 내 사용자의 외부 사이트에 대한 DNS 요청을 확인하여
봇과 멀웨어 그리고 C&C(Command And Control) 서버로의 접속 시도를 사전에 차단하는 DNS 보안 솔루션입니다.
매일 업데이트되는 최대 2조 2천억 건의 DNS 요청에서 얻는 Akamai의 최신 인사이트로 보안을 강화하세요.
Akamai SIA는 단일 대시보드에서 모든 아웃바운드 트래픽과 위협 이벤트, 그 이상에 관한 실시간 인사이트를 보여주며
탐지 엔진 등 멀티 레이어 방어로 사용자 및 디바이스를 보호합니다.

Akamai SIA, 드리는 약속

Akamai SIA는 DNS를 활용해 내부 보안을 강화합니다.
내부 사용자의 멀웨어, 랜섬웨어, 피싱사이트 접근 및 DNS 쿼리로 위장한 내부 기밀 데이터 유출을 차단합니다.
Akamai SIA를 활용해 비업무 사이트 접속을 통제할 수 있으며 웹 접속의 전반적인 모니터링을 누릴 수 있습니다.
또한 클라우드 기반으로 별도 장비 도입 없이 내부 DNS 설정 변경만으로 간편하게 구축이 가능합니다.

Akamai SIA는 기존 보안 스택을 보완하여 전반적인 보안 체계를 강화하는 클라우드 기반의 사전 예방적 보안 레이어를 빠르고 쉽게 구축할 수 있도록 지원합니다.
Akamai EAA와 함께 사용하면 더욱 안전한 원격 접속 서비스를 경험할 수 있습니다.

Akamai SIA 기대효과

글로벌 Akamai 인텔리전스 엣지 플랫폼(Intelligent Edge Platform) DNS 서비스를 기반으로 구축된, Akamai SIA는 빠르게 구성하고 배포할 수 있는 클라우드 보안 웹 게이트웨이입니다.
멀웨어, 렌섬웨어, 피싱 및 DNS 기반 데이터 유출과 같은 표적 위협을 차단하고 위협 이벤트와 사이버 위협에 대한 실시간 인사이트를 제공합니다.
