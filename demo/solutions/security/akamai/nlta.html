Akamai CI & NLTA, 드리는 약속

Akamai CI & NLTA는 방대한 보안 위협 DB 규모와 신속한 업데이트를 자랑합니다. Akamai CI(Client Interlligence)는
Akamai 보안 서비스를 사용 중인 고객 서비스를 공격한 IP 정보로 IP 주소별 행동 패턴을 분석하여
IP 주소에 대한 위험(Risk) 점수를 산정한 자료입니다. Akamai NLTA(Network Layer Threat Actors)는
Akamai 인프라를 공격하거나 스캔 한 이력이 있는 IP 정보로 IP 주소별 행동 패턴을 IP 주소에 대한 위험 (Risk) 점수를 산정한 자료입니다.
