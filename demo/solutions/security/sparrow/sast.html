소스코드 보안 취약점 점검 솔루션

Sparrow SAST

다양한 분석 기술을 바탕으로 소스코드 상의 보안약점을
정확하고 빠르게 검출하는 정적 분석 도구입니다.

자료다운로드
정확하고 빠른 소스코드 취약점 검출

Sparrow SAST는 소프트웨어 개발 단계부터 보안 취약점을 제거하여 보안 취약점 발생 및 침해 사고 발생을 방지합니다.
보안 취약점 사전 제거로 이를 악용한 취약점 발생 리스크가 감소하는 효과를 경험해 보세요.

Sparrow SAST, 드리는 약속

Sparrow SAST는 소스코드 보안 취약점 점검(시큐어 코딩) 도구로
개발 단계부터 소스코드를 분석하여 보안 약점을 검출해 시큐어코딩을 적용할 수 있도록 지원합니다.

다양한 언어 및 점검 항목 지원

① Java, JSP, C/C++, C#, Python, Swift, ASP(.NET), ABAP, Object C, Kotlin 등 20개 이상 언어 지원

② 790여 개의 보안 약점 점검 항목 지원

③ 국내·외 컴플라이언스 및 표준 가이드 준수

빠르고 정확한 분석

① MVC 구조 분석, 연관 파일 분석, 여러 단계의 함수 호출 관계 분석

② 재분석 시 추가·변경·연관된 파일만 분석하여 분석 시간 최소화

③ Sparrow DAST, Sparrow RASP의 연동을 통한 프로그램 연관 취약점 확인

지능형 애플리케이션

① 취약점 원인부터 취약점 발생 지점까지의 과정을 따라가는 이슈 내비게이터 제공

② 검출된 취약점에 대한 실제 소스코드의 자동 수정 제안

③ 위험도 별 취약점 자동 분류

  • 중앙화된 툴 관리를 통해 체커 그룹별로 위험도 및 옵션 관리
타 시스템과의 연동

  • 기간별 분석 결과 이력, 추이 제공
  • 프로젝트, 사용자, 점검 항목별 통계 자료 제공
사용자 맞춤형 보고서
  • 프로젝트 요약정보, 분석 파일 정보, 위험도별 분석 결과 등
    원하는 항목 편집 가능
  • 한글, 워드, 엑셀, PDF 형식의 보고서 제공
분석 결과 이력관리
  • 기존 검출 결과와 비교하여 새롭게 검출된 결과 자동 구분
  • 소스코드 라인이 변경되더라도 기존의 검출 결과 상태 자동 식별
  • 검출 결과 예외 처리 신청/관리자 승인
    프로세스 제공을 통한 부정 사용 방지
다양한 분석 방법
  • 클릭만으로 손쉽게 분석할 수 있는 GUI
  • 배치/스케줄링 분석이 가능한 CLI
  • 개발 IDE 내에 설치되어 분석 및 결과 확인이 가능한 플러그인
  • 별도 클라이언트 프로그램 없이 웹 중앙 관리 시스템에서
    소스코드 드래그 앤 드롭 분석
