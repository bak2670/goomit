웹 애플리케이션 취약점 동적 분석 솔루션

Sparrow DAST

강력한 분석 능력과 높은 사용 편의성을 제공하는
웹 애플리케이션 취약점 동적 분석 도구입니다.

자료다운로드
정확하고 빠른 소스코드 취약점 검출

Sparrow DAST는 개발, 운영 중인 웹 애플리케이션(웹사이트 등) 보안 취약점을 제거하여 보안 취약점 및 침해 사고 발생 방지합니다.
Sparrow DAST를 통해 웹 애플리케이션의 보안 취약점을 사전 제거하여 이를 악용한 취약점 공격 리스크를 감소시킬 수 있습니다.

Sparrow DAST, 드리는 약속

Sparrow DAST는 웹 애플리케이션 보안 취약점을 점검하는 블랙박스(Black Box) 기반의 동적분석 도구입니다.
별도의 설치가 없는 URL 기반으로 취약점 동적 분석이 가능합니다.

최신 웹 애플리케이션 기술 적용

① HTML5, Ajax등 최신 기술들을 사용하는
웹 애플리케이션 분석

② 취약점 공격 과정을 이벤트별로 재현

웹 애플리케이션 취약점 자동 검출

① 웹 애플리케이션의 URL로 부터 포함된 하위 경로들을 자동으로 수집

② 수집된 경로들에 존재하는 보안 취약점 검출

③ 오픈소스 웹 라이브러리 취약점 분석

상호작용 지원
