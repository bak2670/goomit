서버 가상화 기술의 특징

  • VMware HA를 통한 고가용성 제공
  • 가상머신 online중에 다른 host로 마이그레이션 기능 제공
  • 가상머신 online중에 다른 Shard Storage로 마이그레이션 기능 제공
  • 가상머신 성능 데이터 관리 기능 제공
  • 직관적인 UI 기능 제공
  • 통합서버, 네트워크, 스토리지, 가상머신 관리기능 게공
  • 온라인 중에 가상머신의 disk, network card 추가 기능제공
  • 가성머신에 암호화 기능 제공
